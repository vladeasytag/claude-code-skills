# CRM Index

Master directory of all relationships. One markdown file per entity in the subfolders.

## How records work
- **Customers** (`customers/`) — anyone who has purchased.
- **Leads** (`leads/`) — prospects not yet converted.
- **Partners** (`partners/`) — resellers / channel partners.

Naming: `company-or-name-city.md` (e.g. `acme-signs-toronto.md`).
When a lead converts, move the file from `leads/` to `customers/` and note the date.
