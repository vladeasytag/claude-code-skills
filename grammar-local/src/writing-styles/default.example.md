# Writing style: default (example)

Rename this file to `default.md` to activate it (or create `<name>.md` and pass
`--style <name>`). The grammar fixer will nudge corrections toward this voice
*only where it does not conflict with correctness*.

## Voice
- Clear, concise, professional. Prefer short sentences over long ones.
- Plain language over jargon; explain acronyms on first use.
- Warm but not chatty. No filler, no hedging.

## Conventions
- Oxford comma. Numerals for quantities and money.
- Active voice by default.
- Keep the author's original greetings and sign-offs; never invent new ones.

## Do not
- Do not change facts, names, numbers, or amounts.
- Do not add content that was not in the source text.
